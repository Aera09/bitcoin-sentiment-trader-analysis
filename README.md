# Impact of Bitcoin Market Sentiment on Trader Performance

## Assignment Overview
This project analyzes the relationship between Bitcoin market sentiment and trader performance using two datasets:
1. Bitcoin Market Sentiment (Fear–Greed Index)
2. Historical Trader Data from Hyperliquid

The goal is to understand how market psychology (Fear vs Greed) influences trader profitability, risk-taking behavior, and trading patterns.

---

## Objective
The objective of this project is to explore how Bitcoin market sentiment impacts trader performance by:
- Analyzing profitability under different sentiment conditions
- Studying changes in risk behavior and position sizing
- Identifying hidden behavioral patterns among traders
- Providing insights that can support smarter trading strategies

---

## Datasets Used

### 1. Bitcoin Market Sentiment Dataset
- Date
- Classification (Fear, Extreme Fear, Neutral, Greed, Extreme Greed)
- Fear–Greed Index value (0–100)

### 2. Historical Trader Data (Hyperliquid)
- Account
- Coin
- Execution Price
- Size (Tokens & USD)
- Side (Buy/Sell)
- Timestamp
- Closed PnL
- Trade-related attributes

---

## Methodology
1. Data cleaning and preprocessing
2. Date and time alignment between sentiment and trade data
3. Merging daily sentiment data with intraday trades
4. Feature engineering (profit flag, position size, sentiment grouping)
5. Exploratory data analysis
6. Visualization of sentiment-based performance
7. Insight generation and interpretation

---

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Git & GitHub

---

## Key Insights
- Traders tend to take larger positions during Greed phases
- Greed markets show higher volatility and larger losses
- Fear periods encourage more disciplined trading behavior
- Market sentiment influences risk exposure more than win rate

---

## Conclusion
The analysis demonstrates that Bitcoin market sentiment plays a significant role in shaping trader behavior. Understanding sentiment-driven patterns can help traders improve risk management and make more informed trading decisions.
